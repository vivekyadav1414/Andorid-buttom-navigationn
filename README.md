# 🎵 Android Custom Bottom Navigation (Java)

A modern and fully customizable **Bottom Navigation UI** built using **Android (Java + XML)**.
This project demonstrates how to create a **custom-designed bottom navigation bar** instead of using the default `BottomNavigationView`.


https://github.com/user-attachments/assets/06598643-1c56-42f0-8e5e-1d290c38c239


---

## ✨ Features

* 🔥 Custom Bottom Navigation UI (Dark Theme)
* 🎯 Smooth Tab Selection Handling
* 🎨 Dynamic Icon & Text Color Change
* 💡 Highlighted Selected Tab (Bubble Effect)
* ⚡ Lightweight & Easy to Integrate
* 📱 Fully Responsive Layout

---

## 📸 UI Preview

* 3 Tabs: **Local | Search | Online**
* Selected tab shows:

  * Highlight background
  * Purple icon & text
* Unselected tabs remain gray

## 🚀 How It Works

* Custom `LinearLayout` is used as Bottom Navigation
* Each tab is a clickable layout
* On click:

  * Background changes
  * Icon color updates
  * Text color updates
